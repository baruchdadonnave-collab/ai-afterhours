# AI AfterHours

Landing page, pricing page, and thank-you page for a Hebrew-first Voice AI SaaS.

## Stack

- React + Vite + TypeScript
- Tailwind CSS
- shadcn/ui-style local components
- RTL-first dark mode

## Run

```bash
npm install
npm run dev
```

Routes:

- `/` - landing page
- `/pricing` - pricing page with calculator
- `/thank-you` - thank-you page
